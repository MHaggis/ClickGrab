fsquirt.exe

fsquirt.exe is the Bluetooth File Transfer wizard. The Send Files flow opens an Explorer file dialog to choose a file.

windows gui File Explorer GUI

Send files (Explorer dialog)

Try Example
File Explorer GUI

Requires an active Bluetooth radio and the Windows "Bluetooth Support Service" (bthserv) running, plus at least one paired/available device. The Browse… file picker only appears after selecting a target device in the wizard.

  1. Press Win-R

  2. Type fsquirt and press Enter

  3. Choose Send files

  4. Select a paired/available device > Next

  5. Click Browse… and select the requested path

Mitigations:

  • Ensure bthserv is disabled on systems without Bluetooth; user education
